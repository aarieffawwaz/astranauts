# SOP Keselamatan Kerja - PT Pamapersada Nusantara

## 1. Panduan Umum Keselamatan

### 1.1 Prinsip Dasar

- Keselamatan adalah prioritas utama dalam setiap operasi
- Setiap karyawan berhak menghentikan pekerjaan jika kondisi tidak aman
- Gunakan APD (Alat Pelindung Diri) sesuai standar

### 1.2 Highwall Safety

Jika kemiringan tebing melebihi 30°:

1. **Segera evakuasi** semua unit dalam radius 50m
2. **Laporkan** ke Site Supervisor dan Geotechnical Engineer
3. **Pasang barricade** dan rambu bahaya
4. **Jangan lanjutkan operasi** sampai ada inspeksi ulang

### 1.3 Obstacle Detection

Jika terdeteksi halangan pada jarak <10m:

1. **STOP** segera kendaraan
2. **Identifikasi** jenis halangan
3. **Laporkan** ke dispatcher
4. **Tunggu** instruksi lebih lanjut

## 2. Prosedur Darurat

### 2.1 Breakdown

1. Aktifkan hazard light
2. Pasang segitiga pengaman 10m di belakang
3. Hubungi dispatcher via radio
4. Tunggu tim maintenance

### 2.2 Kecelakaan

1. Pastikan keselamatan diri sendiri
2. Hubungi emergency response team
3. Jangan pindahkan korban kecuali bahaya lain
4. Dokumentasikan lokasi kejadian

## 3. Maintenance Preventif

### 3.1 Pengecekan Harian

- Level oli mesin
- Tekanan ban
- Fungsi rem
- Sistem pencahayaan
- Level bahan bakar

### 3.2 Pengecekan Mingguan

- Kondisi filter udara
- Sistem pendingin
- Kondisi V-belt
- Sistem hidrolik
