# Equipment Manual - Alat Berat Tambang

## 1. Haul Truck (HD Series)

### 1.1 Spesifikasi HD-100T

- Kapasitas: 100 ton
- Engine: Diesel 1200 HP
- Max Speed: 60 km/jam
- Fuel Tank: 1500 L
- Konsumsi BBM: 150 L/jam (full load)

### 1.2 Operating Parameters

- Optimal speed: 35-45 km/jam
- Max tilt angle: 15° (loaded)
- Min turning radius: 12m
- Braking distance (40 km/h): 25m

### 1.3 Warning Signs

- **Temperature >90°C**: Stop immediately
- **Oil pressure <30 psi**: Stop immediately
- **Battery <24V**: Check alternator

## 2. Excavator (EX Series)

### 2.1 Spesifikasi PC-4000

- Bucket capacity: 22 m³
- Engine: Diesel 2000 HP
- Max digging depth: 14m
- Cycle time: 25-30 detik

### 2.2 Operating Parameters

- Optimal swing angle: 90°
- Max reach: 18m
- Fuel consumption: 250 L/jam

## 3. Dozer (DZ Series)

### 3.1 Spesifikasi D-375

- Blade width: 4.5m
- Engine: 550 HP
- Operating weight: 65 ton

### 3.2 Operating Parameters

- Max slope: 30° (with load)
- Push distance optimal: 30-50m
