# Mine Plan - Site Berau

## 1. Layout Tambang

### 1.1 Loading Points

- **Loading Point A**: Area utara, untuk overburden removal
- **Loading Point B**: Area tengah, untuk coal getting
- **Loading Point C**: Area selatan, untuk waste dumping

### 1.2 Dumping Points

- **Dumping Point 1**: Disposal area barat
- **Dumping Point 2**: Stockpile ore
- **Dumping Point 3**: Temporary waste area

## 2. Rute Optimal

### 2.1 Rute Hauling Overburden

Loading Point A → Main Haul Road → Dumping Point 1

- Jarak: 2.5 km
- Waktu tempuh optimal: 8 menit
- Max speed: 40 km/jam

### 2.2 Rute Hauling Coal

Loading Point B → Coal Haul Road → Dumping Point 2

- Jarak: 1.8 km
- Waktu tempuh optimal: 6 menit
- Max speed: 35 km/jam

## 3. Target Produksi

### 3.1 Overburden Removal

- Target harian: 150,000 BCM
- Target bulanan: 4.5 juta BCM
- Fleet requirement: 25 trucks + 5 excavators

### 3.2 Coal Getting

- Target harian: 15,000 ton
- Target bulanan: 450,000 ton
- Fleet requirement: 15 trucks + 3 excavators

## 4. Parameter Operasional

### 4.1 Cycle Time Target

- Loading: 2-3 menit
- Hauling: 6-8 menit
- Dumping: 1-2 menit
- Return: 5-7 menit
- Queue: <3 menit

### 4.2 Fuel Efficiency Target

- Overburden: 2.5 L/BCM
- Coal: 3.0 L/ton
